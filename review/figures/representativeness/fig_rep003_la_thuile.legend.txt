FIGURE LEGEND — fig_rep003_la_thuile.png
=========================================

TITLE: La Thuile (n = 252) — sampling ratio per class

DESCRIPTION:
Six-panel grid showing the representativeness of the La Thuile FLUXNET dataset
(n = 252 sites, circa 2007) relative to global vegetated land area, assessed
across the same six classification axes as fig_rep001. La Thuile represented
a substantial expansion over Marconi, more than doubling site count and
extending coverage into tropical and semi-arid regions, though temperate bias
persisted.

PANEL LAYOUT (2 rows × 3 columns, labelled A–F):
  A — Köppen-Geiger climate (Beck 2023, 13-class)
  B — ESA CCI Land Cover v2.1.1 (10-class, high-level)
  C — CGIAR Aridity Index v3.1 (7-class, UNEP scheme)
  D — ESA CCI Biomass v7, 2024 estimate (7-bin hybrid)
  E — TRENDY v14 NEE-IAV (7-bin hybrid)
  F — TRENDY v14 ET-median (7-bin hybrid)

X AXIS: Sampling ratio on log₂ scale; range ±5× (log₂ ±2.32). See Rep001 legend.
Y AXIS: Classes ordered by global land area (largest at top).

GLOBAL LAND AREA DENOMINATORS: Same as fig_rep001.

CLASSIFICATION SCHEMES: Same as fig_rep001.

REPRESENTATIVENESS METRIC: Weighted Jaccard (J). See fig_rep001 legend.

JACCARD VALUES (la_thuile, n = 252):
  A. KG (13-class):            J = 0.292
  B. LULC (10-class HL):       J = 0.553
  C. Aridity (7-class UNEP):   J = 0.531
  D. Biomass (7-bin hybrid):   J = 0.490
  E. NEE-IAV (7-bin hybrid):   J = 0.411
  F. ET-median (7-bin hybrid): J = 0.363

CONTEXT: La Thuile shows improved J over Marconi for all continuous axes,
reflecting expanded tropical and semi-arid coverage. The KG 13-class J (0.292)
remains low, indicating persistent structural gaps in the most granular climate
axis despite the expanded site count.

DATA SOURCES: Same as fig_rep001.
