TITLE: Current FLUXNET (n = 767) vs FLUXNET2015 (n = 212) — sampling ratio per class

DESCRIPTION:
Six-panel overlay grid comparing the class-level sampling ratios of the current
FLUXNET network (n = 767, opaque bars) and FLUXNET2015 (n = 212, semi-transparent
bars at α = 0.42) across the same six classification axes. Within each panel,
paired bars for the same class show whether the current network improved, maintained,
or degraded representativeness relative to FLUXNET2015. Both networks use the same
global KG-land fractions as the denominator; differences reflect changes in the
geographic and environmental distribution of qualifying sites.

PANEL LAYOUT (2 rows × 3 columns, labelled A–F):
  A — Köppen-Geiger climate (Beck 2023, 13-class)
  B — ESA CCI Land Cover v2.1.1 (10-class, high-level)
  C — CGIAR Aridity Index v3.1 (7-class, UNEP scheme)
  D — ESA CCI Biomass v7, 2024 estimate (7-bin hybrid)
  E — TRENDY v14 NEE-IAV (7-bin hybrid)
  F — TRENDY v14 ET-median (7-bin hybrid)

NOTE: Panels D–F show 7-bin hybrid classification (legacy); see fig_rep007/008
for the updated 18-bin hybrid results.

X AXIS: Sampling ratio on log₂ scale; range ±5× (log₂ ±2.32). See Rep001 legend.
Y AXIS: Classes ordered by global land area (largest at top).
LEGEND: Opaque bars = Current (n=767); semi-transparent bars = FLUXNET2015 (n=212).

GLOBAL LAND AREA DENOMINATORS: Same as fig_rep001.

CLASSIFICATION SCHEMES: Same as fig_rep001.

REPRESENTATIVENESS METRIC: Weighted Jaccard (J). J values for both networks shown
  in top-right corner of each panel (current above, 2015 below).

JACCARD VALUES (current_767 / fluxnet2015):
  A. KG (13-class):            J = 0.373 (current) / 0.365 (2015)  +0.008
  B. LULC (10-class HL):       J = 0.556 (current) / 0.582 (2015)  −0.026
  C. Aridity (7-class UNEP):   J = 0.667 (current) / 0.598 (2015)  +0.069
  D. Biomass (7-bin hybrid):   J = 0.616 (current) / 0.540 (2015)  +0.076
  E. NEE-IAV (7-bin hybrid):   J = 0.486 (current) / 0.477 (2015)  +0.009
  F. ET-median (7-bin hybrid): J = 0.433 (current) / 0.366 (2015)  +0.067

NOTE ON LULC: The current network has slightly lower LULC J than FLUXNET2015.
This reflects the inclusion of additional sites in underrepresented vegetation
classes (e.g. croplands, grasslands) that are abundant globally but were better
proportioned in the curated FLUXNET2015 subset.
